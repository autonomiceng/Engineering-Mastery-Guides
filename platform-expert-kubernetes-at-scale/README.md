# From Linux Admin to Platform Expert: Kubernetes at Scale

An advanced training book for experienced Linux and cloud engineers who want to operate Kubernetes as a platform, not just consume it as a product.

> [!NOTE]
> This book is an LLM-assisted draft under active manual review. Treat all explanations and labs as review-required material.

## Start Here

- Read the [INTRODUCTION.md](./INTRODUCTION.md)
- Navigate with [TABLE_OF_CONTENTS.md](./TABLE_OF_CONTENTS.md)
- Use chapter-local labs under `src/chapter_XX/`

## Reader Profile

| This book assumes you already know | This book focuses on |
| --- | --- |
| Linux operations, networking, SSH, cloud primitives, and basic Kubernetes usage | Control plane internals, distributed systems behavior, RKE2 architecture, GitOps, eBPF networking, CSI mechanics, scheduling, and day-2 operations |

## Design Principles

- No beginner padding
- Failure-mode-first explanations
- Linux and kernel mechanics where they matter
- Concrete cloud versus on-prem tradeoffs
- Labs that produce real filesystem artifacts, not hand-wavy pseudo-code
